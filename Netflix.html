Data description

The variables we will use in the Process, Analysis and Share phases of the project are the following:

Netflix Movies and TV Shows

  • Type: Movie or TV Show
  • Title: Name of the show.
  • Director: Director(s) of the show.
  • Casting: Principal actors and actresses of the show.
  • Country: Country where the show is from.
  • Date_added: Date when the show was added to Netflix (US).
  • Release_year: Year when the show was released.
  • Rating: What audience is the show aimed to.
      Movies:
    • G: General audiences.
    • PG: Parental guidance suggested.
    • PG-13: Parents strongly cautioned.
    • R: Restricted.
    • NC-17: Adults only.
      TV Shows:
    • TV-Y: Children under 7.
    • TV-Y7: Intended to children ages 7.
    • TV-Y7 FV: Children ages 7 with Fantasy Violence.
    • TV-G: General audiences.
    • TV-PG: Parental guidance suggested.
    • TV-14: Parents strongly cautioned.
    • TV-MA: Mature audience only.
  • Duration: How long the show lasts. Minutes in movies and seasons in tv shows.
  • Listed_in: Classification of the show: Documentaries, Comedies, Drama, etc
  • Description: Short description of the show.

Prepare

As we claimed, although our data was previously extracted and prepared due to using imported DFs from Kaggle. We had to make sure there were no NULLs or dirty data in general.

We realized our data had some NULL values in the "director", "casting" and "country" columns so we wrote a query to create a new depurated table without these NULL values.

								
SELECT
	*
INTO
	NETFLIX.dbo.netflix_cleaned
FROM
	NETFLIX.dbo.netflix_titles
WHERE
	director IS NOT NULL
AND
	casting IS NOT NULL
AND
	country IS NOT NULL

This query returns a table with 5336 rows, which means that 3471 rows of our original table with 8807 rows contained NULL values

Process

Having all of the data in our database we proceeded to write SQL queries to extract information and be able to answer the previous questions.

Average BPM

								
SELECT
	AVG(CAST (Beats_Per_Minute AS Float)) as BPM_Average

	top50_2019

Analyze

What artists appear the most?

As we could see in the table of frequencies the most popular artist from 2019 was Ed Sheeran, followed by Billie Eilish and Ariana Grande.

Which is the most predominant genre in the top songs of the chart?

The most predominant genre in the top 25 songs was Pop, because the most repeated genres were variants of itself like Dance Pop, Panamanian Pop or just Pop itself.

Is an easier-to-dance song most likely to appear at the top?

It seems like an easier-to-dance song is not likely to appear in the top songs of the chart, but the difference is so small that we would need a bigger dataset to be confident about it.

Is there any song repeated in both charts?

Indeed, as surprising as it seems there is one song that got to the top 50 chart in both years 2019 and 2021 and it was Someone You Loved by Lewis Capaldi.
